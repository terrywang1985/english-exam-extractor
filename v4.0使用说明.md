## 🚀 使用方法

### 方式一：打印完整试卷（学生答题用）

1. 打开试卷页面（如 `试卷.html`）
2. 点击浏览器扩展图标
3. 点击 **"导出完整试卷"** 按钮
4. 下载的HTML文件包含：
   - ✅ 所有题目（完形填空、阅读理解等）
   - ✅ 所有选项
   - ✅ 所有答案（黄色背景标注）
5. 打开HTML文件，按 `Ctrl+P` 打印

**适用场景：**
- 学生自主练习
- 课后作业
- 需要参考答案的学习材料

---

### 方式二：分别打印试卷和答案（考试用）⭐ 推荐

#### 步骤1：导出完整试卷
1. 打开试卷页面
2. 点击扩展图标
3. 点击 **"导出完整试卷"**
4. 保存为：`英语试卷_完整版_2025-11-04.html`

#### 步骤2：导出答案页
1. 继续在同一页面
2. 点击扩展图标
3. 点击 **"导出答案页"**
4. 保存为：`英语试卷_答案页_2025-11-04.html`

#### 步骤3：打印
1. 打开完整试卷HTML → 打印 → 给学生答题
2. 打开答案页HTML → 打印 → 教师留存批改

**适用场景：**
- 课堂测验
- 模拟考试
- 正式考试
- 需要分开管理试卷和答案

## ⚙️ 打印设置

### 推荐设置
1. 打开HTML文件
2. 按 `Ctrl+P` 或点击"打印"
3. **重要设置**：
   - ✅ 取消勾选"页眉和页脚"
   - ✅ 勾选"背景图形"（显示颜色）
   - ✅ 纸张：A4
   - ✅ 方向：纵向
   - ✅ 边距：默认

### 为什么要勾选"背景图形"？
- 完整试卷：显示答案的黄色背景
- 答案页：显示答案的绿色高亮
- 题型标题：显示蓝色背景

## 📊 题型支持

| 题型 | 完整试卷 | 答案页 |
|-----|---------|--------|
| 单项选择 | ✅ 题目+选项+答案 | ✅ 题号+答案字母 |
| 完形填空 | ✅ 文章+下划线+选项表格+答案 | ✅ 题号+答案字母 |
| 选词填空 | ✅ 选项框+文章+下划线+答案 | ✅ 题号+答案词组 |
| 首字母填空 | ✅ 文章+首字母+下划线+答案 | ✅ 题号+完整单词 |
| 阅读理解 | ✅ 文章+题目+选项+答案 | ✅ 题号+答案字母 |
| 其他题型 | ✅ 完整题目+答案 | ✅ 题号+答案文本 |

## 💡 使用技巧

### 技巧1：考试前分开打印
```
1. 考试前一天：
   - 导出完整试卷 → 打印50份 → 给学生
   - 导出答案页 → 打印1份 → 教师留存

2. 考试当天：
   - 学生用完整试卷答题
   - 教师用答案页快速批改
```

### 技巧2：课后作业带答案
```
1. 导出完整试卷（含答案）
2. 打印给学生课后练习
3. 学生可以自主核对答案
```

### 技巧3：制作错题集
```
1. 导出答案页
2. 打印多份
3. 学生答错后，对照答案页标注
4. 整理成错题本
```

## 🔍 常见问题

### Q1: 为什么没有PDF了？
**A:** PDF生成有中文显示问题，HTML格式更稳定、更完美。

### Q2: 答案页和完整试卷有什么区别？
**A:** 
- **完整试卷**：包含题目、选项、答案，适合学生练习或参考
- **答案页**：只有题号和答案，适合教师批改

### Q3: 答案页能单独用吗？
**A:** 可以，但需要配合完整试卷使用。答案页只有答案，没有题目内容。

### Q4: 打印时答案没有颜色？
**A:** 打印设置中需要勾选"背景图形"选项。

### Q5: 如何快速批改？
**A:** 
1. 学生在完整试卷上答题
2. 教师对照答案页逐题核对
3. 答案页按题型分组，查找方便
