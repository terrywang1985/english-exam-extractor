# 🚀 快速上手指南

## 安装扩展

1. 打开 `chrome://extensions/`
2. 找到"英语试卷提取器"
3. 点击刷新按钮 🔄
4. 确保已启用

---

## 两种使用方式

### 📄 方式一：完整试卷（学生练习用）

```
打开试卷 → 点击扩展 → 导出完整试卷 → 打印
```

**包含内容：**
- ✅ 所有题目
- ✅ 所有选项
- ✅ 所有答案（黄色标注）

**适合：** 课后作业、自主学习

---

### 📋 方式二：试卷+答案分开（考试用）⭐ 推荐

```
打开试卷 → 点击扩展 → 导出完整试卷 → 保存
           ↓
         点击扩展 → 导出答案页 → 保存
           ↓
        分别打印
```

**完整试卷：** 给学生答题  
**答案页：** 教师批改用

**适合：** 测验、考试

---

## 打印设置

按 `Ctrl+P` 打印时：

✅ **取消** "页眉和页脚"  
✅ **勾选** "背景图形"  
✅ 纸张：A4  
✅ 方向：纵向

## 常见场景

### 场景1：课堂测验
1. 导出完整试卷 → 打印30份 → 学生答题
2. 导出答案页 → 打印1份 → 批改用

### 场景2：课后作业
1. 导出完整试卷（含答案）
2. 打印给学生
3. 学生可自主核对

### 场景3：期末复习
1. 导出多套试卷
2. 答案页整理成册
3. 方便学生查阅

## 核心优势

| 功能 | 传统方式 | 本工具 |
|-----|---------|--------|
| 试卷提取 | 手动复制粘贴 | ✅ 一键导出 |
| 格式保留 | 需要重新排版 | ✅ 完美保留 |
| 答案分离 | 手动删除答案 | ✅ 单独答案页 |
| 打印质量 | 依赖原网页 | ✅ 优化排版 |
| 中文显示 | PDF可能乱码 | ✅ HTML完美 |
