# 英语试卷提取器 Chrome 扩展

这是一个Chrome扩展程序，可以自动提取网页中的英语试卷内容和答案，并生成PDF文件供下载。

## 功能特点

- 自动识别试卷页面中的题目内容
- 提取题目的答案信息
- 将试卷内容整理成PDF格式
- 提供下载提示，用户确认后下载PDF文件
- 支持多种题型（单选题、完形填空、阅读理解等）

## 安装步骤

1. 打开Chrome浏览器，进入 `chrome://extensions/` 页面
2. 打开右上角的"开发者模式"开关
3. 点击"加载已解压的扩展程序"按钮
4. 选择本文件夹（english-exam-extractor）
5. 扩展安装完成

## 使用方法

1. 在Chrome浏览器中打开包含英语试卷的网页
2. 点击浏览器工具栏中的扩展图标（绿色E图标）
3. 点击"提取试卷并生成PDF"按钮
4. 等待程序自动提取试卷内容并生成PDF
5. 根据浏览器提示保存PDF文件

## 技术说明

- 使用Content Script提取网页中的试卷内容
- 使用jsPDF库生成PDF文件
- 支持提取题目、选项和答案
- 答案在PDF中以红色标识

## 适用范围

本扩展主要适用于以下格式的试卷网页：
- 包含.paper-title类的试卷标题
- 使用h3.ques-type标记的题型标题
- 使用li.QUES_LI标记的题目元素
- 答案包含在.sanwser类的元素中

## 注意事项

- 本扩展主要针对特定格式的英语试卷网页
- 如果网页结构不同，可能需要调整内容提取逻辑
- 生成的PDF文件会自动下载到浏览器的默认下载目录
- 如遇到问题，请检查网页是否符合上述结构要求
